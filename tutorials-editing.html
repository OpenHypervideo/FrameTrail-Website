Tutorials - Editing Hypervideos

Entering Edit Mode

To start editing, click the Edit button in the top-right corner of the player. Depending on how FrameTrail is deployed:
  • Server mode: Log in with your username and password.
  • Local folder mode: Enter a display name — no password required.
  • Guest mode (in-memory): Log in as a guest. Changes can be exported as JSON but are not saved automatically.
Once in edit mode, the interface expands to show the timeline editor below the video player.

Creating a New Hypervideo

  1. In edit mode, click "New Hypervideo" in the titlebar.
  2. Give your hypervideo a title and description.
  3. Choose a video source. FrameTrail supports:
    • HTML5 video — upload a file or link to a direct video URL (MP4, HLS)
    • YouTube — paste a YouTube URL
    • Vimeo — paste a Vimeo URL
    • Time-based container — a virtual video with a set duration but no video source, useful for purely annotation-driven content
  4. Click Create. FrameTrail will open the new hypervideo in the editor.
[ Screenshot: The "New Hypervideo" dialog with video source options ]

The Timeline Editor

The timeline editor appears below the video player in edit mode. It shows tracks for each type of content you can add:

  • Overlays — elements placed on top of the video at a specific time and position
  • Code Snippets — JavaScript code triggered at a specific timestamp
  • Annotations — supporting materials shown alongside (not on top of) the video
  • Subtitles — VTT subtitle tracks synchronized with the video

To add a resource to the timeline, drag it from the Resource Panel on the left onto the appropriate track. You can then resize and reposition items by dragging their handles.

[ Screenshot: The timeline editor with overlays and annotation tracks visible ]

Working with Overlays

Overlays are placed on top of the video and are visible to all users. Only users with the admin role can add or edit overlays.

Adding an Overlay

  1. Open the Overlays tab in the editor panel.
  2. Drag a resource from the Resource Panel onto the overlay track in the timeline, or click "Add Overlay" and select a resource.
  3. Set the start time and duration by dragging the item's edges on the timeline.
  4. In the video area, drag the overlay to its position and resize it as needed.

Custom HTML Overlays

In the "Custom HTML" tab, you can create overlays containing any HTML markup — useful for styled text, buttons, or embedded content not covered by built-in resource types.

Video Links

A special overlay type — Video Links — creates a clickable area on the video that navigates to another hypervideo project when clicked. This allows you to build branching or interconnected hypervideo experiences.

[ Screenshot: An overlay positioned on top of the video, with the position/size handles visible ]

Working with Annotations

Annotations are user-specific supporting materials displayed outside the video area in the Layout Areas. Every user manages their own set of annotations independently.

  1. Open the Annotations tab in the editor panel.
  2. Drag a resource from the Resource Panel onto the annotation track, or click "Add Annotation".
  3. Set the start time and duration on the timeline.

You can also browse and reuse annotations created by other users: open the "Annotations of other Users" tab for an overview of all annotation sets for the current hypervideo.

Code Snippets

Code Snippets let you define JavaScript code that is executed automatically at a specific point in the video timeline. This allows advanced interactions such as controlling the page, triggering external APIs, or dynamically modifying the FrameTrail interface.

  1. Open the Code Snippets tab.
  2. Click "Add Code Snippet" and set the timestamp at which it should fire.
  3. Write your JavaScript in the built-in code editor.

Code Snippets are shared across all users (like Overlays) and can only be added or edited by admin users.

Subtitles

FrameTrail supports subtitle tracks in WebVTT (.vtt) format, synchronized with the video.

  1. Open the Subtitles tab in the editor panel.
  2. Upload a .vtt file or add one via URL from the Resource Manager.
  3. The subtitle track will appear on the timeline and be rendered as an interactive transcript in the Layout Area if configured.

Saving your Work

Server Mode

Click Save in the titlebar. Changes are written directly to the server's _data/ directory.

Local Folder Mode

Click Save. Changes are written directly to the _data/ folder on your disk via the File System Access API.

Guest / In-Memory Mode

The Save button is disabled. Use Save As to export a JSON snapshot of your hypervideo, which can be reloaded later.

Sharing your Hypervideo

Once saved, you can share your hypervideo in two ways:

  • Direct URL: Copy the URL from the browser — it contains a deep link to the specific hypervideo.
  • Embed: Use the embed button in the player to get an <iframe> snippet you can paste into any webpage.
